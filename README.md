# 💍 Sneha & Ezaz — Wedding Invitation

A beautiful, animated wedding invitation website for **Sneha & Ezaz**, to be celebrated on **2nd May 2026** at **Katiahat**.

## ✨ Features

- Animated floating petals background
- Live countdown timer to the wedding day
- Elegant floral corner decorations
- Smooth scroll-reveal animations
- Wedding details cards (time, venue, dress code)
- Quranic verse section
- RSVP form
- Fully responsive (mobile & desktop)

## 🚀 Hosting on GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save** — your site will be live at `https://<your-username>.github.io/<repo-name>/`

## 📁 Files

```
index.html   ← The invitation page (all-in-one)
README.md    ← This file
```

## 🛠 Customisation

All content is in `index.html`. To update:
- **Names** — search for `Sneha` and `Ezaz`
- **Date/Time** — update the countdown target date and the details section
- **Venue** — update the venue card
- **Parents' names** — update the "Our Families" section
- **Colors** — tweak the CSS variables at the top of the `<style>` block
